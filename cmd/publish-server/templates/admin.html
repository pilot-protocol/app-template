Rotate a publisher key

Re-point an app's owning publisher key (e.g. after a key loss or a handoff). Opens a re-signed catalogue PR on the platform repo. After it merges, the new owner must publish an update signed with the new key before existing installs re-validate.

The new owner generates it with pilotctl appstore gen-key and gives you the public line.
